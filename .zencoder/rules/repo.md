# Repository Assistance Notes

- **Framework**: Vite + React + TypeScript + Tailwind CSS
- **UI Libraries**: Swiper carousel
- **Primary Folder Structure**:
  - `src/components` for reusable UI components
  - `src/pages` for page-level components
  - `src/store` for state management utilities
  - `src/api` for API client abstractions
- **Styling**: Tailwind utilities via `index.css`
- **Build Tools**: Vite with TypeScript configurations in `tsconfig.*`
- **Common Tasks**:
  - Run dev server: `npm run dev`
  - Build: `npm run build`
  - Preview: `npm run preview`
- **Notable Dependencies**: Swiper (ensure CSS imports align with installed version)
